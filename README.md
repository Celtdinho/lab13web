# Praktikum 14 – Pencarian Data dengan PHP & MySQL

## Deskripsi
Aplikasi web sederhana berbasis PHP dan MySQL yang menerapkan fitur **login**, **CRUD data barang**, serta **pencarian data** menggunakan klausa `WHERE` dan `LIKE` sesuai dengan modul Praktikum 14.

## Fitur Aplikasi
- Autentikasi Login Admin
- Menampilkan data barang
- Menambah, mengubah, dan menghapus data barang
- Pencarian data barang berdasarkan nama
- Antarmuka sederhana dan interaktif

## Teknologi yang Digunakan
- PHP
- MySQL
- HTML & CSS
- XAMPP / Apache

## Struktur Database
- **users** (id, username, password)
- **data_barang** (id, nama_barang, kategori, harga)

## Cara Menjalankan Aplikasi
1. Aktifkan Apache dan MySQL
2. Import database ke MySQL
3. Simpan folder project ke dalam direktori `htdocs`
4. Akses melalui browser:
http://localhost/praktikum14/auth/login.php
